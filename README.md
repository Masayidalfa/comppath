## Instalasi

**Persyaratan:**

* Git
* Node.js (dengan npm)
* Composer

### Klone Repository
```bash
git clone https://github.com/Masayidalfa/comppath.git
```
### Masuk ke Direktori Proyek
```bash
cd comppath
```
### Instalasi Dependensi
```bash
npm install && npm run build
composer run dev
```
